# 센서 사용
- 포토레지스터 ( 빛의 세기에 따라 저항값이 달라짐)
```
void setup(){
  Serial.begin(9600);
}

void loop(){
  Serial.print(analogRead(A0));
}

```

- 포텐셜미터 (손으로 돌리는 가변 저항)
```
void setup(){
  Serial.begin(9600);
}

void loop(){
  Serial.print(analogRead(A0));
}
```
